# 🐱 CatPhotoApp

## 📖 Project Description

CatPhotoApp is my first HTML project created while learning the **Responsive Web Design Certification** from freeCodeCamp.

This project was developed by following the step-by-step lessons provided in the course. Each lesson introduced a new HTML concept, allowing me to gradually build a complete webpage while understanding the fundamentals of web development.

The project demonstrates how a simple webpage can be structured using HTML elements such as headings, paragraphs, links, images, lists, figures, and semantic tags.

---

## 🎯 Purpose of the Project

The purpose of this project was to:

* Learn the basics of HTML5.
* Understand webpage structure.
* Practice writing clean and semantic HTML.
* Build confidence in creating web pages from scratch.
* Prepare for more advanced topics such as CSS, Responsive Design, and JavaScript.

---

## 🚀 What I Learned

During the development of this project, I learned:

### 1. HTML Document Structure

* `<!DOCTYPE html>`
* `<html>`
* `<head>`
* `<body>`

### 2. Headings and Text Content

* `<h1>`
* `<h2>`
* `<h3>`
* `<p>`

### 3. Hyperlinks

* Creating clickable links using `<a>`
* Opening links in new tabs with `target="_blank"`

### 4. Images

* Adding images using `<img>`
* Improving accessibility using the `alt` attribute

### 5. Lists

* Unordered Lists (`<ul>`)
* Ordered Lists (`<ol>`)
* List Items (`<li>`)

### 6. Semantic HTML

* `<main>`
* `<section>`
* `<figure>`
* `<figcaption>`
* `<footer>`

### 7. Text Formatting

* `<strong>`
* `<em>`
